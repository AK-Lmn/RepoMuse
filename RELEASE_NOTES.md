# RepoMuse v1.0.0

RepoMuse is a native Android app that turns public GitHub repositories into portfolio-ready case studies.

## Main features

- GitHub repository import and authentication
- AI-generated pitches, problem statements, features, challenges, and resume bullets
- Local persistence and optional Firestore sync
- PDF export for generated case studies

## Setup and security

Provide your own local `.env` and `app/google-services.json` files before building. These files contain project-specific configuration and are ignored by Git. No real API keys, Firebase configuration, or other credentials are included in this repository.
